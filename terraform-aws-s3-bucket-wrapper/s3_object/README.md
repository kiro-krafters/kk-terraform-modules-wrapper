# s3_object

## How to use this module:

### aws s3 object module usage with the required input variables:

```terraform
module "s3_object" {
  source         = "../"
  prefix_company = "ch"
  lob            = "telesales"
  prefix_region  = "use1"
  application    = "connect"
  env            = "dev"
  create_object  = true
  bucket         = "test"
  key            = "test_key"
}

```
